# Prompts

Production-grade, copy-paste prompts for common engineering tasks. Each defines a **Role,
Objective, Constraints, Output Format, and Quality Requirements**, and enforces the toolkit's
architecture and security standards. Fill the `{{…}}` placeholders before use.

- [feature_generator.md](feature_generator.md) — build a full feature
- [architecture_review.md](architecture_review.md) — review structure & coupling
- [security_audit.md](security_audit.md) — OWASP MASVS audit
- [code_review.md](code_review.md) — prioritized PR review
- [performance_audit.md](performance_audit.md) — diagnose & fix performance
- [websocket_design.md](websocket_design.md) — design a realtime feature
- [api_integration.md](api_integration.md) — integrate REST/GraphQL
- [bug_investigation.md](bug_investigation.md) — root-cause a bug
- [refactor_feature.md](refactor_feature.md) — safe behavior-preserving refactor
- [generate_tests.md](generate_tests.md) — generate a test suite

These prompts pair with the [agents](../agents/), [workflows](../workflows/), and
[checklists](../checklists/) they reference.
